Tests was created in C# with NUnit framework.
To make sure they will work install Visual Studio Community 2019 (with .NET packages) from: https://visualstudio.microsoft.com/thank-you-downloading-visual-studio/?sku=Community&rel=16
and the latest version of Google Chrome from:
https://www.google.com/intl/pl_pl/chrome/

How to run tests:
1. Download project from https://github.com/ggqkamilg/ng_kg/archive/master.zip
2. Extract project.
3. Open 'ng_kg.sln'.
4. Open test explorer (Ctrl + E, T or View -> Test Explorer).
5. Run all tests ('Run All tests' button or Ctrl + R, A) or run them separately by selecting specific test and clicking 'Play' button.
